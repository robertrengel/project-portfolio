Sobre mi

Hola, mi nombre es Robert Rengel y soy un desarrollador web apasionado. Mi experiencia en el campo me permite crear soluciones personalizadas para cada proyecto, utilizando mis habilidades en CSS para dar estilo y diseño, Javascript para añadir interacción y dinamismo, HTML como base para cualquier sitio web, Git para el control de versiones, Python y Django para el desarrollo back-end y MySQL y Postgres para el manejo de bases de datos. Siempre estoy buscando aprender más y estoy emocionado de trabajar en nuevos proyectos y desafíos.
